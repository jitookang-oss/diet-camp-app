참여자 등록 — 한 명씩 추가

참여자를 한 명씩 직접 입력해서 등록하는 방법입니다.

1
오른쪽 상단 "+ 참여자 추가" 버튼 클릭 등록 모달창이 열립니다.
2
이름 입력 예: 홍길동
3
전화번호 입력 (하이픈 없이) 예: 01012345678
4
"등록하기" 버튼 클릭 성공하면 목록에 바로 추가됩니다.
같은 전화번호로 다시 등록하면 정보가 덮어씌워지므로 중복 등록이 되지 않습니다.
등록 후 자동으로 되는 것
  • 다음 날부터 해당 번호로 카카오 알림톡이 자동 발송됩니다.
  • 참여자 목록에 바로 표시됩니다.
4
참여자 등록 — 엑셀 일괄 업로드

여러 명을 한 번에 등록할 때 사용합니다.

엑셀 파일 형식

첫 번째 행(헤더)에 아래 열 이름이 있어야 합니다.

이름 전화번호
홍길동 01012345678
김영희 01098765432
이철수 01055556666
열 이름은 아래 단어 중 하나면 자동으로 인식됩니다.
이름: 이름 · 성명 · Name  |  전화번호: 전화번호 · 핸드폰 · 휴대폰 · 연락처 · Phone

업로드 방법

1
"📊 엑셀 업로드" 버튼 클릭
2
"파일 선택" 클릭 → .xlsx 또는 .csv 파일 선택 파일을 선택하면 미리보기가 자동으로 표시됩니다.
3
미리보기에서 내용 확인 각 행의 상태 열에 ✓가 표시되어야 정상입니다. "오류"가 있으면 전화번호 형식을 확인하세요.
4
"N명 등록하기" 버튼 클릭 오류 항목이 있으면 버튼이 비활성화됩니다.
전화번호는 반드시 숫자만 입력하세요 (하이픈 자동 제거). 예: 01012345678
엑셀에서 전화번호 셀의 서식이 "숫자"로 설정되어 있으면 앞자리 0이 사라질 수 있습니다. 셀 서식을 "텍스트"로 변경하세요.

참여자 입장의 전체 흐름

참여자가 앱을 처음 사용할 때부터 끝까지의 과정입니다.

① 첫 접속 (온보딩)

1
메인 화면에서 이름 + 생년월일 입력 예: 홍길동, 1980-05-15
2
신체정보 입력 성별, 키, 현재 몸무게 → BMI·목표체중 자동 계산
3
건강상태 입력 카카오 알림 수신 번호, 복용약, 질환 여부
4
갱년기 증상 선택 (여성만)

② 1주차 설문 (체질 분석)

25문항 설문 (식사·활동·마음 3개 파트) → 체질 유형 분석 → 결과 화면

  • 습관정착형 ✨
  • 혈당불안정형 🍽️
  • 대사저하형 🚶
  • 스트레스과식형 🧘
  • 복합형 ⚡

③ 매일 체크인 (2~12주)

알림톡 버튼 클릭 → 체크인 완료 (하루 3번 기회)

④ 주차별 설문 (매주)

대시보드 → 이번 주 설문 → 몸무게 + 25문항 입력 → 진행 현황 업데이트

⑤ 12주차 최종 설문 + 결과

  • 12주 전후 3M 점수 비교
  • 몸무게 변화 그래프
  • PDF 결과지 다운로드

자주 묻는 질문 (FAQ)
Q. 참여자가 전화기를 바꿨을 때 기존 데이터를 복원할 수 있나요?

현재는 기기에 저장된 데이터를 자동 복원하는 기능이 없습니다. 참여자가 동일한 이름과 생년월일로 다시 입력하면 새로 시작됩니다. 이 기능은 추후 업데이트 예정입니다.

Q. 알림톡이 발송되지 않아요.

① 참여자의 전화번호가 등록되어 있는지 확인하세요.
② 카카오톡이 설치되어 있어야 수신됩니다.
③ 발송 시각(오전 8시, 오후 12:30, 오후 7시)이 지났는지 확인하세요.

Q. 참여자가 이미 체크인했다고 나오는데 실제로 안 했어요.

같은 날 같은 유형의 링크를 한 번 클릭하면 완료 처리됩니다. 실수로 클릭한 경우에도 완료로 처리되며, 수동 취소는 현재 지원하지 않습니다.

Q. 엑셀 업로드 시 앞자리 0이 사라져요.

엑셀에서 전화번호가 입력된 셀의 서식을 텍스트로 변경한 후 다시 저장하세요.
방법: 해당 열 선택 → 셀 서식 → 텍스트

Q. 캠프 시작일을 변경하고 싶어요.

개발자에게 연락하면 변경할 수 있습니다. 현재 설정: 2026년 6월 1일

Q. 관리자 비밀번호를 변경하고 싶어요.

개발자에게 연락하면 변경할 수 있습니다.
